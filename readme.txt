== Description ==

This plugin lets you change your login logo on Wordpress. It may be usefull to change it for your customers.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `lm-login-logo` directory to your `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Visit the 'LM login logo' menu item in your admin sidebar
